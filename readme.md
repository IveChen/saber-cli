### commands
#### init
> create project in current fold
```
sbr init
```


#### create <projectName>
> create project in current fold
```
sbr create <projectName>
```


#### run [pageName]
> run project with [pageName] page
```
sbr run [pageName]
//sbr run user/list
```

#### page <pageName>
> add <pageName> page
```
sbr page <pageName>
//sbr page user/list
```

#### build
> build project
```
    sbr build
```

#### version
> show saber-cli version
```
    sbr --version
    sbr -V
```
