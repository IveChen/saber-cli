### install
```
npm install saber-cli -g
```

### commands
#### init
> create project in current fold
```
sbr init
```


#### create <projectName>
> create project in current fold
```
sbr create <projectName>
//eg: sbr create hello-world
```


#### run [pageName]
> run project with [pageName] page
```
sbr run [pageName]
//eg: sbr run user/list
```

#### page <pageName>
> add <pageName> page
```
sbr page <pageName>
//eg: sbr page user/list
```

#### build
> build project
```
sbr build
```

#### version
> show saber-cli version
```
sbr --version
sbr -V
```
